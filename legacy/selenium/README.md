# Legacy Selenium Whatsapp-Bot

This folder documents the **original** [SeifReda30/Whatsapp-Bot](https://github.com/SeifReda30/Whatsapp-Bot) project (Jan 2024):

- Python + Selenium + Excel bulk outreach
- No web UI, no API, no multi-tenant support
- Chrome profile for WhatsApp Web login

The new **WhatsApp Sender SaaS** in the monorepo root replaces this approach with Baileys background workers and a full dashboard.

To reference the old code, see the git mirror at:
